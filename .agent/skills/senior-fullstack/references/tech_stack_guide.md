# Tech Stack Guide

## Overview
Recommended technology stack for modern fullstack applications.

## Frontend
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Components**: Radix UI / Shadcn UI
- **State Management**: React Context, Jotai, or Zustand

## Backend
- **Runtime**: Node.js / Bun
- **Framework**: Next.js API Routes or Express
- **API Style**: GraphQL (Apollo/Yoga) or REST
- **Auth**: NextAuth.js / Clerk

## Database
- **Primary**: PostgreSQL
- **ORM**: Prisma / Drizzle
- **Hosting**: Neon / Supabase

## DevOps & Cloud
- **CI/CD**: GitHub Actions
- **Deployment**: Vercel / Railway
- **Monitoring**: Sentry / Logtail
